# UnivSearch
University Search Website



- Search and result page implemented using node.js ,express.js and request
- University's web addresses are attached to every university name



1. Run the following commands to execute the website

   In your local terminal 
     - git clone https://github.com/ajay-droid/UnivSearch.git
     - npm install
     - node app,js
   In your browser 
     - localhost:3000
     
     
2. Test Cases - look for following in the search menu in the website

  - India (This will succesfully load all the universities in the API)
  - England (This will pop an error because no content exists)
